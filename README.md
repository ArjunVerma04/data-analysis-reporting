# Data Analysis & Reporting System

A data analysis project that cleans business datasets, generates performance reports, and visualizes key metrics using Python and SQL.

## Features
- Data cleaning and preprocessing
- Weekly and monthly performance reporting
- SQL-based data analysis
- KPI trend visualization
- Automated report generation

## Tech Stack
- Python
- Pandas
- SQLite (SQL)
- Matplotlib

## Project Structure
data-analysis-reporting/
│
├── data/
├── reports/
├── sql/
├── src/
├── requirements.txt
└── README.md

## Run Project
pip install -r requirements.txt
python src/analysis.py

## Output
- Console-based business performance summary
- Revenue trend chart saved in reports folder
